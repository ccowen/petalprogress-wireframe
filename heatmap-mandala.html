Click Predictions

  • #1 target: Check-in card (89% predicted click rate)
  • #2 target: Theme swatches (72% — high visual appeal)

Frustration Hotspots

  • Mandala non-interactivity: Largest element feels clickable but isn't
  • No undo on check-in: Single-click with no confirmation
  • Settings auto-save ambiguity: No visible save state or confirmation
  • Stats lack actionable context (what's "good"?)
  • Avatar dropdown has no visual cue that it's interactive

Recommendations

  • Add hover/click feedback to mandala petals (tooltip with date/habit info)
  • Add undo toast after check-in: "Marked complete · Undo"
  • Show save confirmation in settings (auto-save badge or subtle toast)
  • Move Share/Shop into toolbar or right panel for discoverability
  • Add a dropdown caret to avatar for affordance
  • Add motivational context to stats: "You're on track!" / streak fire
